1 Introduction

These are some rough notes as we try to see if we can arrive at a monotonic entropy formula for the parabolic \(p\)-harmonic equation coupled with Ricci flow similar to the one shown by Perelman for the heat equation.

2 The heat equation case

First we review the set up and the results for the heat equation case. My main references for this are (Chow, Lu, and Ni 2006) and (Chow et al. 2007). The setup is as follows. Let \((M,g)\) be a closed \(n\)-dimensional Riemmanian manifold, \(f\) be a function on \(M\) and \(\tau > 0\). Let \(u = (4\pi\tau)^{-n/2}e^{-f}\). Then we define the entropy by

\[\mathcal{W}(g,f,\tau) := \int_M \left(\tau\left(R + |\nabla f|^2\right) + (f-n)\right)ud\mu.\]

Consider the following system of equations

\[ \frac{\partial}{\partial t} g_{ij} = -2R_{ij}, \](2.1)

\[ \frac{\partial f}{\partial t} = -\Delta f - R + |\nabla f|^2 + \frac{n}{2\tau}, \](2.2)

\[ \frac{\partial \tau}{\partial t} = -1. \](2.3)

That is, we have \(f(t)\) satisfying a backward heat-type equation coupled with the Ricci flow. Then when this system is satisfied, we have the following entropy monotonicity result.

Theorem 1 (Theorem 5.23 of (Chow, Lu, and Ni 2006) or Theorem 6.4 of (Chow et al. 2007)). If \((M,g(t))\) and \(f(t)\) satisfy (2.1), (2.2), and (2.3), then

\[ \frac{d}{dt}\mathcal{W}(g(t),f(t),\tau(t)) = 2\tau\int_M \left| R_{ij} + \nabla_i\nabla_j f - \frac{1}{2\tau}g_{ij}\right|^2 ud\mu \geq 0. \](2.4)

There are two proofs of this result. The first is to derive it from a similar monotonicity formula for the energy \(\mathcal{F}:= \int_M (R + |\nabla f|^2)e^{-f}d\mu\) under a gradient flow (that differs from the Ricci flow by diffeomorphisms, see Chapter 2.4 of (Chow, Lu, and Ni 2006) and subsequently Exercise 6.7 of (Chow et al. 2007)), and the second is to use an evolution equation for the adjoint operator to the heat equation (Proposition 5.24 of (Chow, Lu, and Ni 2006) and/or Pages 227-229 of (Chow et al. 2007)).

3 Kotschwar-Ni’s work

In (Kotschwar and Ni 2009), an entropy monotonicity formula is shown for the \(p\)-harmonic equation in the static case (that is, not coupled to the Ricci flow). Let \(p > 1\)

\[v^{p-1} = \left(\pi p^{\ast p -1}p\right)^{-n/p}\frac{\Gamma(n/2+1)}{\Gamma(n/p^\ast + 1}e^{-\varphi}t^{-n/p},\]

where \(p^\ast = \frac{p}{p-1}\).

We define the static \(p\)-entropy as

\[\mathcal{W}_p(\varphi, t) := \int_M (t|\nabla\varphi|^2 + \varphi - n)v^{p-1}d\mu.\]

Then when \(v^{p-1}\) satisfies the following parabolic \(p\)-harmonic equation,

\[ \frac{\partial}{\partial t}v^{p-1} = (p-1)^{p-1}\text{div}(|\nabla v|^{p-2}\nabla v), \](3.1)

we have the following entropy monotonicity formula.

Theorem 2 (Theorem 6.13 of (Kotschwar and Ni 2009)). Let \(v\) be a positive solution to (3.1) above normalized so that \(\int_M v^{p-1}d\mu = 1\). Then

\[ \frac{d}{dt}\mathcal{W}_p(\varphi, t) = -tp \int_M \left(\left||\nabla\varphi|^{p-2}\nabla_i\nabla_j\varphi - \frac{1}{tp}a_{ij}\right|_A^2 + |\nabla \varphi|^{2p-4}R_{ij}\varphi_i\varphi_j\right)v^{p-1}d\mu, \](3.2)

where \(A^{ij} = g^{ij} + \frac{p-2}{|\nabla \varphi|^2}\varphi^i\varphi^j\) and \(a_{ij} = g_{ij} - \frac{p-2}{p-1}\frac{\varphi_i\varphi_j}{|\nabla\varphi|^2}\) is the matrix inverse of \(A^{ij}\).

Note that in terms of \(\varphi\), it is easy enough to show that

\[\begin{aligned} \nabla v^{p-1} &= -v^{p-1}\nabla\varphi, \\ \frac{\partial v^{p-1}}{\partial t} &= -v^{p-1}\frac{\partial\varphi}{\partial t} - v^{p-1}\frac{n}{pt},\end{aligned}\]

So using this, (3.1) above becomes

\[ \frac{\partial\varphi}{\partial t} = |\nabla\varphi|^{p-2}\Delta\varphi - |\nabla\varphi|^p + (p-2)|\nabla\varphi|^{p-4}\left\langle\nabla|\nabla\varphi|^2,\nabla\varphi\right\rangle - \frac{n}{pt}. \](3.3)

So when \(p = 2\), this reduces to the forward version of the heat equation considered in (2.2) above, which also explains the change in sign for the static \(p\)-entropy monotonicity formula above in (3.2).

4 The \(p\)-harmonic equation coupled with Ricci flow

Because we want to be able to invoke the static \(p\)-entropy monotonicity formula (3.2) above, we’ll couple (3.3) with the backward Ricci flow

\[ \frac{\partial}{\partial t} g_{ij} = 2R_{ij}. \](4.1)

Also, for convenience, we denote the right hand side of (3.3) by \(\Lambda_p(\varphi)\). Then we consider the backward Ricci flow coupled with the following \(p\)-heat type equation

\[ \frac{\partial\varphi}{\partial t} = \Lambda_p(\varphi) + R|\nabla\varphi|^{p-2}. \](4.2)

Then we want to see if we can arrive at a monotonicity formula for the moving \(p\)-entropy

\[\tilde{\mathcal{W}_p}(\varphi, t) := \int_M \left(t\left(R + |\nabla\varphi|^p\right) + \varphi - n\right)v^{p-1}d\mu.\]

Note that I’m using the \(\tilde{}\) symbol to differentiate it from the static \(p\)-entropy just to keep things clear.

First, we have the following formulas for the variation of the metric inverse, the volume form and the scalar curvature under the backward Ricci flow respectively.

\[ \frac{\partial}{\partial t}g^{ij} = -2g^{ik}g^{j\ell}R_{k\ell}, \](4.3)

\[ \frac{\partial R}{\partial t} = -\Delta R - 2|\text{Rc}|^2, \](4.4)

\[ \frac{\partial}{\partial t}d\mu = Rd\mu. \](4.5)

Then, instead of requiring that \(\int_M v^{p-1}d\mu = 1\), we introduce the constraint that \(\frac{d}{dt}\int_M v^{p-1}d\mu = 0\). This means

\[\begin{aligned} 0 &= \frac{d}{dt}\int_M v^{p-1}d\mu \\ &= \int_M \partial_t (v^{p-1}) d\mu + \int_M v^{p-1}\partial_t (d\mu) \\ &= \int_M v^{p-1}\left(-\frac{\partial\varphi}{\partial t} - \frac{n}{pt}\right)d\mu + \int_M Rv^{p-1}d\mu.\end{aligned}\]

Which implies that

\[\frac{\partial\varphi}{\partial t} + \frac{n}{pt} = R.\]

Now we proceed with the main calculation.

\[\begin{aligned} \frac{d}{dt}\tilde{\mathcal{W}_p} &= \int_M \partial_t\left(t\left(R + |\nabla\varphi|^p\right) + \varphi - n\right)v^{p-1}d\mu + \int_M \left(t\left(R + |\nabla\varphi|^p\right) + \varphi - n\right)\partial_t\left(v^{p-1}d\mu\right) \\ &= \int_M \partial_t\left(t\left(R + |\nabla\varphi|^p\right) + \varphi - n\right)v^{p-1}d\mu \\ &= \int_M\left(R + \partial_tR + |\nabla\varphi|^p + t\partial_t(|\nabla\varphi|^p) + \partial_t\varphi\right)v^{p-1}d\mu \end{aligned}\](4.6)

where the second equality is because we have the constraint \(\frac{d}{dt}\int_Mv^{p-1}d\mu = 0\).

Then note that

\[\begin{aligned} \partial_t(|\nabla\varphi|^p) &= \frac{p}{2}|\nabla\varphi|^{p-2}\partial_t(g^{ij}\nabla_i\varphi\nabla_j\varphi) \\ &= \frac{p}{2}|\nabla\varphi|^{p-2}\partial_t(g^{ij})\nabla_i\varphi\nabla_j\varphi + p|\nabla\varphi|^{p-2}\langle\nabla\partial_t\varphi,\nabla\varphi\rangle \\ &= -p|\nabla\varphi|^{p-2}g^{ik}g^{j\ell}R_{k\ell}\nabla_i\varphi\nabla_j\varphi + p|\nabla\varphi|^{p-2}\langle\nabla\Lambda_p(\varphi),\nabla\varphi\rangle + p|\nabla\varphi|^{p-2}\langle\nabla(R|\nabla\varphi|^{p-2}),\nabla\varphi\rangle \\ &= -p|\nabla\varphi|^{p-2}\varphi^k\varphi^{\ell}R_{k\ell} + p|\nabla\varphi|^p\langle\nabla\Lambda_p(\varphi),\nabla\varphi\rangle \\ &\qquad + p|\nabla\varphi|^{2p-4}\langle\nabla R,\nabla\varphi\rangle + p(p-2)|\nabla\varphi|^{2p-6}R\langle\nabla|\nabla\varphi|^2,\nabla\varphi\rangle \end{aligned}\](4.7)

where we have used (4.3) and (4.2) in the third equality above. So substituting this into the end of (4.6) and continuing, we have

\[\begin{aligned} \frac{d}{dt}\tilde{\mathcal{W}_p} &= \int_M\left(R + \partial_tR + |\nabla\varphi|^p + t\partial_t(|\nabla\varphi|^p) + \partial_t\varphi\right)v^{p-1}d\mu \\ &= \int_M \left(|\nabla\varphi|^p + tp|\nabla\varphi|^p\langle\nabla\Lambda_p(\varphi),\nabla\varphi\rangle + \Lambda_p(\varphi)\right)v^{p-1}d\mu \\ &\qquad + \int_M \left(R - t\Delta R - 2t|\text{Rc}|^2 -tp|\nabla\varphi|^{p-2}\varphi^k\varphi^{\ell}R_{k\ell} \right. \\ &\qquad\qquad \left. + tp|\nabla\varphi|^{2p-4}\langle\nabla R,\nabla\varphi\rangle + tp(p-2)|\nabla\varphi|^{2p-6}R\langle\nabla|\nabla\varphi|^2,\nabla\varphi\rangle + R|\nabla\varphi|^{p-2}\right)v^{p-1}d\mu \\ &= -tp \int_M \left(\left||\nabla\varphi|^{p-2}\nabla_i\nabla_j\varphi - \frac{1}{tp}a_{ij}\right|_A^2 + |\nabla \varphi|^{2p-4}R_{ij}\varphi_i\varphi_j\right)v^{p-1}d\mu \\ &\qquad -tp\int_M\left(-\frac{1}{tp}R + \frac{1}{p}\Delta R + \frac{2}{p}|\text{Rc}|^2 + |\nabla\varphi|^{p-2}\varphi^k\varphi^{\ell}R_{k\ell} \right. \\ &\qquad\qquad \left. - |\nabla\varphi|^{2p-4}\langle\nabla R,\nabla\varphi\rangle - (p-2)|\nabla\varphi|^{2p-6}R\langle\nabla|\nabla\varphi|^2,\nabla\varphi\rangle - \frac{1}{tp}R|\nabla\varphi|^{p-2}\right)v^{p-1}d\mu \end{aligned}\](4.8)

where we have used Kotschwar-Ni’s result in the last equality above.

When \(p=2\), (4.8) above reduces to

\[\begin{aligned} \frac{d}{dt}\tilde{\mathcal{W}_2} &= -2t \int_M \left(\left|\nabla_i\nabla_j\varphi - \frac{1}{2t}g_{ij}\right|^2 + R_{ij}\varphi_i\varphi_j -\frac{1}{t}R + \frac{1}{2}\Delta R + |\text{Rc}|^2 + \varphi^k\varphi^{\ell}R_{k\ell} - \langle\nabla R,\nabla\varphi\rangle\right)ud\mu \\ &= -2t\int_M\left(|\text{Rc}|^2 + R_{ij}\varphi_i\varphi_j + \varphi^k\varphi^{\ell}R_{k\ell} -\frac{1}{t}R + \frac{1}{2}\Delta R - \langle\nabla R,\nabla\varphi\rangle + |\nabla\nabla\varphi|^2 - \frac{1}{t}\Delta\varphi + \frac{n}{4t^2}\right)ud\mu. \end{aligned}\](4.9)

where here \(u = (4\pi\tau)^{-n/2}e^{-\varphi}\). Assuming \(M\) is closed so we can apply integration by parts, and using \(\nabla u = -u\nabla\varphi\), we have

\[-\int_M \langle\nabla R,\nabla\varphi\rangle ud\mu = \int_M \langle \nabla R,\nabla u\rangle d\mu \\ = -\int_M \Delta R ud\mu,\]

so (4.9) becomes

\[\frac{d}{dt}\tilde{\mathcal{W}_2} = -2t\int_M\left(|\text{Rc}|^2 + R_{ij}\varphi_i\varphi_j + \varphi^k\varphi^{\ell}R_{k\ell} -\frac{1}{t}R - \frac{1}{2}\Delta R + |\nabla\nabla\varphi|^2 - \frac{1}{t}\Delta\varphi + \frac{n}{4t^2}\right)ud\mu. \](4.10)

which unfortunately differs from the expected formula of

\[\int_M\left|R_{ij} + \nabla_i\nabla_j \varphi - \frac{1}{2t}g_{ij}\right|^2ud\mu = \int_M\left(|\text{Rc}|^2 + 2g^{ik}g^{j\ell}R_{k\ell}\nabla_i\nabla_j\varphi - \frac{1}{t}R + |\nabla\nabla\varphi|^2 - \frac{1}{t}\Delta\varphi + \frac{n}{4t^2}\right)ud\mu.\]

Note that if instead we chose

\[\frac{\partial\varphi}{\partial t} = \Lambda_p(\varphi) + R,\]

then (4.8) would instead become

\[\begin{aligned} \frac{d}{dt}\tilde{\mathcal{W}_p} &= -tp \int_M \left(\left||\nabla\varphi|^{p-2}\nabla_i\nabla_j\varphi - \frac{1}{tp}a_{ij}\right|_A^2 + |\nabla \varphi|^{2p-4}R_{ij}\varphi_i\varphi_j\right)v^{p-1}d\mu \\ &\qquad -tp\int_M\left(-\frac{2}{tp}R + \frac{1}{p}\Delta R + \frac{2}{p}|\text{Rc}|^2 + |\nabla\varphi|^{p-2}\varphi^k\varphi^{\ell}R_{k\ell} - |\nabla\varphi|^{p-2}\langle\nabla R,\nabla\varphi\rangle\right)v^{p-1}d\mu\end{aligned}\]

which still reduces to (4.10) when taking \(p = 2\).
